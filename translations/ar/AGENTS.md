## أوامر الإعداد

### المتطلبات المسبقة
- بايثون 3.12 أو أعلى
- حساب GitHub (لنماذج GitHub - المستوى المجاني)
- اشتراك Azure (اختياري، لخدمات Azure AI)

### الإعداد الأولي

1. **استنساخ أو فورك للمستودع:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # أو
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **إنشاء وتفعيل بيئة افتراضية بايثون:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # على ويندوز: venv\Scripts\activate
   ```

3. **تثبيت التبعيات:**
   ```bash
   pip install -r requirements.txt
   ```

4. **إعداد متغيرات البيئة:**
   ```bash
   cp .env.example .env
   # عدل ملف .env بمفاتيح API ونقاط النهاية الخاصة بك
   ```

### متغيرات البيئة المطلوبة

بالنسبة لـ **نماذج GitHub (مجاني)**:
- `GITHUB_TOKEN` - رمز الوصول الشخصي من GitHub

بالنسبة لـ **خدمات Azure AI** (اختياري):
- `PROJECT_ENDPOINT` - نقطة نهاية مشروع Microsoft Foundry
- `AZURE_OPENAI_API_KEY` - مفتاح API الخاص بـ Azure OpenAI
- `AZURE_OPENAI_ENDPOINT` - عنوان نقطة نهاية Azure OpenAI
- `AZURE_OPENAI_CHAT_DEPLOYMENT_NAME` - اسم النشر لنموذج الدردشة
- `AZURE_OPENAI_EMBEDDING_DEPLOYMENT_NAME` - اسم نشر التضمينات
- تكوينات Azure إضافية كما هو موضح في `.env.example`

## سير عمل التطوير

### تشغيل دفاتر جوبتر

يحتوي كل درس على دفاتر جوبتر متعددة لأُطُر مختلفة:

1. **تشغيل جوبتر:**
   ```bash
   jupyter notebook
   ```

2. **انتقل إلى مجلد درس** (مثلاً `01-intro-to-ai-agents/code_samples/`)

3. **فتح وتشغيل دفاتر الملاحظات:**
   - `*-semantic-kernel.ipynb` - إطار عمل Semantic Kernel
   - `*-autogen.ipynb` - إطار عمل AutoGen
   - `*-python-agent-framework.ipynb` - إطار عمل Microsoft Agent Framework (بايثون)
   - `*-dotnet-agent-framework.ipynb` - إطار عمل Microsoft Agent Framework (.NET)
   - `*-azureaiagent.ipynb` - خدمة وكلاء Azure AI

### العمل مع أُطُر مختلفة

**Semantic Kernel + نماذج GitHub:**
- مستوى مجاني متاح مع حساب GitHub
- جيد للتعلم والتجريب
- نمط الملفات: `*-semantic-kernel*.ipynb`

**AutoGen + نماذج GitHub:**
- مستوى مجاني متاح مع حساب GitHub
- قدرات تنسيق للوكلاء المتعددين
- نمط الملفات: `*-autogen.ipynb`

**إطار عمل Microsoft Agent (MAF):**
- أحدث إطار عمل من Microsoft
- متوفر بايثون و .NET
- نمط الملفات: `*-agent-framework.ipynb`

**خدمة وكلاء Azure AI:**
- يتطلب اشتراك Azure
- ميزات جاهزة للإنتاج
- نمط الملفات: `*-azureaiagent.ipynb`

## تعليمات الاختبار

هذا مستودع تعليمي يحتوي على أمثلة تعليمية بدلاً من رمز إنتاج مع اختبارات آلية. للتحقق من إعدادك والتغييرات:

### الاختبار اليدوي

1. **اختبر بيئة بايثون:**
   ```bash
   python --version  # يجب أن يكون 3.12 أو أعلى
   pip list | grep -E "(autogen|semantic-kernel|azure-ai)"
   ```

2. **اختبر تشغيل دفاتر الملاحظات:**
   ```bash
   # تحويل الدفتر إلى سكريبت وتشغيله (يختبر الاستيرادات)
   jupyter nbconvert --to script <lesson-folder>/code_samples/<notebook>.ipynb --stdout | python
   ```

3. **تحقق من متغيرات البيئة:**
   ```bash
   python -c "import os; from dotenv import load_dotenv; load_dotenv(); print('✓ GITHUB_TOKEN' if os.getenv('GITHUB_TOKEN') else '✗ GITHUB_TOKEN missing')"
   ```

### تشغيل دفاتر الملاحظات الفردية

افتح دفاتر الملاحظات في جوبتر ونفذ الخلايا بالتسلسل. كل دفتر مستقل ويحتوي على:

- تنفيذات نموذجية للوكلاء
- المخرجات المتوقعة في خلايا ماركداون
