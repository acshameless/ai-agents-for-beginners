# Структура на тетрадката

Jupyter тетрадките са JSON документи със следната обща структура:

- `nbformat` и `nbformat_minor`
- `metadata`
- `cells` (списък от markdown и кодови клетки)

When editing `.ipynb` files programmatically:

- Запазете `nbformat` и `nbformat_minor` от шаблона.
- Дръжте `cells` като подреден списък; не пренареждайте освен ако не е умишлено.
- За кодови клетки задайте `execution_count` на `null`, когато е неизвестно.
- За кодови клетки задайте `outputs` като празен списък при създаване на шаблон.
- За markdown клетки запазете `cell_type="markdown"` и `metadata={}`.

Предпочитайте използването на вградените шаблони или `new_notebook.py` (например, `$CODEX_HOME/skills/jupyter-notebook/scripts/new_notebook.py`) вместо ръчното създаване на суров JSON за тетрадки.
