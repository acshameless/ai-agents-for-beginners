# নোটবুক কাঠামো

Jupyter নোটবুকগুলো এই উচ্চ-স্তরের আকারের JSON ডকুমেন্ট:

- `nbformat` এবং `nbformat_minor`
- `metadata`
- `cells` (markdown এবং কোড সেলগুলির একটি তালিকা)

When editing `.ipynb` files programmatically:

- টেম্পলেট থেকে `nbformat` এবং `nbformat_minor` বজায় রাখুন.
- `cells`-কে একটি ক্রমানুচিত তালিকা হিসেবে রাখুন; ইচ্ছাকৃত না হলে পুনরায় ক্রমবদল করবেন না.
- কোড সেলগুলোর ক্ষেত্রে, অজানা থাকলে `execution_count` কে `null` সেট করুন.
- কোড সেলগুলোর ক্ষেত্রে, স্ক্যাফোল্ডিং করার সময় `outputs` কে একটি খালি তালিকা হিসেবে সেট করুন.
- মার্কডাউন সেলগুলোর জন্য, `cell_type="markdown"` এবং `metadata={}` রাখুন.

হাত দিয়ে কাঁচা নোটবুক JSON লেখার বদলে বান্ডেল করা টেমপ্লেট বা `new_notebook.py` থেকে স্ক্যাফোল্ডিং করাকে অগ্রাধিকার দিন (উদাহরণস্বরূপ, `$CODEX_HOME/skills/jupyter-notebook/scripts/new_notebook.py`).
