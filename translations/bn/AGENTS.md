# AGENTS.md

## প্রকল্পের ওভারভিউ

এই রেপোজিটরিতে "শুরু করা শিক্ষার্থীদের জন্য AI এজেন্ট" রয়েছে - একটি ব্যাপক শিক্ষা কোর্স যা AI এজেন্ট তৈরির জন্য প্রয়োজনীয় সব কিছু শেখায়। এই কোর্সে ১৫+টি পাঠ রয়েছে যা ফান্ডামেন্টালস, ডিজাইন প্যাটার্ন, ফ্রেমওয়ার্ক, এবং AI এজেন্টের প্রোডাকশন ডিপ্লয়মেন্ট কভার করে।

**মূল প্রযুক্তি:**
- পাইথন ৩.১২+
- ইন্টারেক্টিভ শিক্ষার জন্য জুপিটার নোটবুক
- AI ফ্রেমওয়ার্ক: সেমান্টিক কার্নেল, অটোজেন, মাইক্রোসফট এজেন্ট ফ্রেমওয়ার্ক (MAF)
- আজুর AI সার্ভিস: মাইক্রোসফট ফাউন্ড্রি, আজুর AI এজেন্ট সার্ভিস
- গিটহাব মডেল মার্কেটপ্লেস (ফ্রি টিয়ার উপলব্ধ)

**আর্কিটেকচার:**
- পাঠভিত্তিক স্ট্রাকচার (০০-১৫+ ডিরেক্টরিস)
- প্রতিটি পাঠে থাকে: README ডকুমেন্টেশন, কোড স্যাম্পল (জুপিটার নোটবুক), ও ছবি
- স্বয়ংক্রিয় অনুবাদ ব্যবস্থার মাধ্যমে মাল্টি-ভাষা সমর্থন
- প্রতিটি পাঠে একাধিক ফ্রেমওয়ার্ক অপশন (সেমান্টিক কার্নেল, অটোজেন, আজুর AI এজেন্ট সার্ভিস)

## সেটআপ কমান্ডস

### প্রাক-প্রয়োজনীয়তা
- পাইথন ৩.১২ বা তার উপরে
- গিটহাব অ্যাকাউণ্ট (গিটহাব মডেলস - ফ্রি টিয়ারের জন্য)
- আজুর সাবস্ক্রিপশন (ঐচ্ছিক, আজুর AI সার্ভিসের জন্য)

### প্রাথমিক সেটআপ

১. **রেপোজিটরি ক্লোন অথবা ফর্ক করুন:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # অথবা
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

২. **পাইথন ভার্চুয়াল পরিবেশ তৈরি ও সক্রিয় করুন:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # উইন্ডোজে: venv\Scripts\activate
   ```

৩. **ডিপেনডেন্সি ইনস্টল করুন:**
   ```bash
   pip install -r requirements.txt
   ```

৪. **এনভায়রনমেন্ট ভেরিয়েবল সেট করুন:**
   ```bash
   cp .env.example .env
   # আপনার API কী এবং এন্ডপয়েন্টগুলি সহ .env সম্পাদনা করুন
   ```

### প্রয়োজনীয় এনভায়রনমেন্ট ভেরিয়েবলসমূহ

**গিটহাব মডেলস (ফ্রি) এর জন্য:**
- `GITHUB_TOKEN` - গিটহাব থেকে ব্যক্তিগত অ্যাক্সেস টোকেন

**আজুর AI সার্ভিস (ঐচ্ছিক):**
- `PROJECT_ENDPOINT` - মাইক্রোসফট ফাউন্ড্রি প্রজেক্ট এন্ডপয়েন্ট
- `AZURE_OPENAI_API_KEY` - আজুর OpenAI API কী
- `AZURE_OPENAI_ENDPOINT` - আজুর OpenAI এন্ডপয়েন্ট URL
- `AZURE_OPENAI_CHAT_DEPLOYMENT_NAME` - চ্যাট মডেলের ডিপ্লয়মেন্ট নাম
- `AZURE_OPENAI_EMBEDDING_DEPLOYMENT_NAME` - এমবেডিংস ডিপ্লয়মেন্ট নাম
- অতিরিক্ত আজুর কনফিগারেশন `.env.example` এ দেওয়া আছে

## ডেভেলপমেন্ট ওয়ার্কফ্লো

### জুপিটার নোটবুক চালানো

প্রতিটি পাঠে বিভিন্ন ফ্রেমওয়ার্কের জন্য একাধিক জুপিটার নোটবুক রয়েছে:

১. **জুপিটার শুরু করুন:**
   ```bash
   jupyter notebook
   ```

২. **পাঠের ডিরেক্টরিতে যান** (যেমন: `01-intro-to-ai-agents/code_samples/`)

৩. **নোটবুক খুলুন ও চালান:**
   - `*-semantic-kernel.ipynb` - সেমান্টিক কার্নেল ফ্রেমওয়ার্ক ব্যবহার
   - `*-autogen.ipynb` - অটোজেন ফ্রেমওয়ার্ক ব্যবহার
   - `*-python-agent-framework.ipynb` - মাইক্রোসফট এজেন্ট ফ্রেমওয়ার্ক (পাইথন)
   - `*-dotnet-agent-framework.ipynb` - মাইক্রোসফট এজেন্ট ফ্রেমওয়ার্ক (.NET)
   - `*-azureaiagent.ipynb` - আজুর AI এজেন্ট সার্ভিস

### বিভিন্ন ফ্রেমওয়ার্ক নিয়ে কাজ

**সেমান্টিক কার্নেল + গিটহাব মডেলস:**
- গিটহাব অ্যাকাউণ্ট দিয়ে ফ্রি টিয়ার
- শেখা ও পরীক্ষণের জন্য ভালো
- ফাইল প্যাটার্ন: `*-semantic-kernel*.ipynb`

**অটোজেন + গিটহাব মডেলস:**
- গিটহাব অ্যাকাউণ্ট দিয়ে ফ্রি টিয়ার
- মাল্টি-এজেন্ট অর্কেস্ট্রেশন ক্ষমতা
- ফাইল প্যাটার্ন: `*-autogen.ipynb`

**মাইক্রোসফট এজেন্ট ফ্রেমওয়ার্ক (MAF):**
- মাইক্রোসফটের সর্বশেষ ফ্রেমওয়ার্ক
- পাইথন ও .NET উভয় ভ্যারিয়েন্টে উপলব্ধ
- ফাইল প্যাটার্ন: `*-agent-framework.ipynb`

**আজুর AI এজেন্ট সার্ভিস:**
- আজুর সাবস্ক্রিপশন প্রয়োজন
- প্রোডাকশন-রেডি ফিচার
- ফাইল প্যাটার্ন: `*-azureaiagent.ipynb`

   - গিটহাব টোকেনের যথাযথ অনুমতি থাকতে হবে

৩. **প্যাকেজ কনফ্লিক্ট:**
   - নতুন ভার্চুয়াল পরিবেশ ব্যবহার করুন
   - পৃথক প্যাকেজের চেয়ে `requirements.txt` থেকে ইনস্টল করুন
   - কিছু নোটবুকের মার্কডাউন সেলে অতিরিক্ত প্যাকেজের উল্লেখ থাকতে পারে

৪. **আজুর সার্ভিস:**
   - আজুর AI সার্ভিসের জন্য সক্রিয় সাবস্ক্রিপশন বাধ্যতামূলক
   - কিছু ফিচার অঞ্চলভিত্তিক
   - গিটহাব মডেলসের ফ্রি টিয়ারের সীমাবদ্ধতা রয়েছে

### ডিপেনডেন্সি

`requirements.txt` থেকে প্রধান প্যাকেজ:
- `autogen-agentchat`, `autogen-core`, `autogen-ext` - অটোজেন ফ্রেমওয়ার্ক
- `semantic-kernel` - সেমান্টিক কার্নেল ফ্রেমওয়ার্ক
- `agent-framework` - মাইক্রোসফট এজেন্ট ফ্রেমওয়ার্ক
- `azure-ai-inference`, `azure-ai-projects` - আজুর AI সার্ভিস
- `azure-search-documents` - আজুর AI সার্চ ইন্টিগ্রেশন
- `chromadb` - RAG উদাহরণে ভেক্টর ডাটাবেস
- `chainlit` - চ্যাট UI ফ্রেমওয়ার্ক
- `browser_use` - এজেন্টের জন্য ব্রাউজার অটোমেশন
- `mcp[cli]` - মডেল কনটেক্সট প্রোটোকল সাপোর্ট
- `mem0ai` - এজেন্টের জন্য মেমোরি ম্যানেজমেন্ট
