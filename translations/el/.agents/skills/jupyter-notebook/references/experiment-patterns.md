# Πρότυπα Πειραμάτων

Χρησιμοποιήστε αυτή τη δομή για διερευνητική και πειραματική εργασία:

- Τίτλος και στόχος: διατυπώστε την ερώτηση και τα κριτήρια επιτυχίας.
- Ρύθμιση και αναπαραγωγιμότητα: εισάγετε μόνο ό,τι χρειάζεστε, ορίστε μια τιμή seed νωρίς, και κρατήστε τη διαμόρφωση σε ένα μικρό κελί.
- Σχέδιο: καταγράψτε τις υποθέσεις, τις σάρωσεις παραμέτρων και τα μετρικά πριν τρέξετε τον κώδικα.
- Ελάχιστη βάση αναφοράς: ξεκινήστε με το μικρότερο εκτελέσιμο παράδειγμα και επιβεβαιώστε ότι εκτελείται από την αρχή έως το τέλος πριν προσθέσετε πολυπλοκότητα.
- Αποτελέσματα και σημειώσεις: συνοψίστε τα ευρήματα σε markdown κοντά στον σχετικό κώδικα και καταγράψτε τα βασικά μετρικά σε ένα μικρό λεξικό ή σε μια πινακοειδή δομή.
- Επόμενα βήματα: αποφασίστε αν θα συνεχίσετε, θα αλλάξετε πορεία ή θα σταματήσετε, και καταγράψτε ιδέες για επόμενα βήματα ως σύντομες κουκκίδες.
