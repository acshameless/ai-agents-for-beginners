# Δομή Σημειωματαρίου

Τα Jupyter notebooks είναι έγγραφα JSON με αυτήν την υψηλού επιπέδου δομή:

- `nbformat` και `nbformat_minor`
- `metadata`
- `cells` (λίστα κελιών markdown και κώδικα)

Όταν επεξεργάζεστε αρχεία `.ipynb` προγραμματιστικά:

- Διατηρήστε `nbformat` και `nbformat_minor` από το πρότυπο.
- Κρατήστε τα `cells` ως διατεταγμένη λίστα; μην αναδιατάσσετε εκτός αν είναι σκόπιμο.
- Για τα κελιά κώδικα, ορίστε το `execution_count` σε `null` όταν είναι άγνωστο.
- Για τα κελιά κώδικα, ορίστε το `outputs` σε μια κενή λίστα κατά τη δημιουργία του σκελετού.
- Για τα markdown κελιά, διατηρήστε `cell_type="markdown"` και `metadata={}`.

Προτιμήστε τη δημιουργία σκελετού από τα ενσωματωμένα πρότυπα ή το `new_notebook.py` (για παράδειγμα, `$CODEX_HOME/skills/jupyter-notebook/scripts/new_notebook.py`) αντί να γράφετε χειροκίνητα το ακατέργαστο JSON του σημειωματαρίου.
