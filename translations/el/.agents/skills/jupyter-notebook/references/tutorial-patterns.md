# Πρότυπα Μαθημάτων

Χρησιμοποιήστε αυτήν τη δομή για διδασκαλία και οδηγούς βήμα-βήμα:

- Audience, prerequisites, and learning goals: αναφέρετε για ποιον προορίζεται, απαριθμήστε τι θα πρέπει ήδη να γνωρίζουν, και δηλώστε τι θα μπορούν να κάνουν στο τέλος.
- Outline: παρέχετε ένα σύντομο αριθμημένο περίγραμμα ώστε οι αναγνώστες να μπορούν να το επισκοπήσουν.
- Step-by-step flow: συνδυάστε μια σύντομη εξήγηση σε markdown με ένα μικρό κελί κώδικα που εκτελείται αυτόνομα και μια σύντομη ερμηνεία του αποτελέσματος.
- Exercises: συμπεριλάβετε τουλάχιστον μία άσκηση που ενισχύει την κύρια ιδέα και παρέχετε ένα σκελετό απάντησης στο επόμενο κελί.
- Pitfalls and extensions: επισημάνετε ένα κοινό λάθος και πώς να το διορθώσετε, και προτείνετε μια προαιρετική επέκταση για τους ενδιαφερόμενους αναγνώστες.
