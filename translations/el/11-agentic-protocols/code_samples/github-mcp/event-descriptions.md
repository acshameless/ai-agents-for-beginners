## Όνομα Εκδήλωσης: Multi-Agent API με LangGraph και Azure Cosmos DB

## Περιγραφή

Η άνοδος των πολυ-πρακτορικών εφαρμογών AI μετασχηματίζει τον τρόπο που δημιουργούμε ευφυή συστήματα — αλλά πώς τα αρχιτεκτονίζετε για πραγματική κλιμάκωση και απόδοση; Σε αυτή τη συνεδρία, θα κάνουμε μια βαθιά βουτιά σε μια παραγωγικής τάξης πολυ-πρακτορική εφαρμογή κατασκευασμένη με LangGraph για ορχήστρωση πρακτόρων, FastAPI για το επίπεδο API, και Azure Cosmos DB ως τη ραχοκοκαλιά για διαχείριση κατάστασης, αποθήκευση διανυσμάτων και συναλλακτικά δεδομένα.

Μέσα από μια λεπτομερή περιήγηση στον κώδικα, θα δείτε πώς να σχεδιάσετε και να υλοποιήσετε μια ροή εργασίας καθοδηγούμενη από agents που ενσωματώνει απρόσκοπτα retrieval-augmented generation (RAG), επιμονή μνήμης, και δυναμικές μεταβάσεις κατάστασης. Θα καλύψουμε:

Agent collaboration with LangGraph for structured reasoning
Real-time chat history storage using Azure Cosmos DB - the same database that powers the chat history in ChatGPT, the fastest-growing AI agent application in history
Vector search for knowledge retrieval with Cosmos DB's native embeddings support
FastAPI’s async capabilities to keep interactions responsive and scalable
By the end of this session, you’ll have a clear blueprint for building and deploying your own scalable, cloud-native multi-agent applications that harness the power of modern AI and cloud infrastructure. Whether you're an AI engineer, cloud architect, or Python developer, this talk will equip you with practical insights and battle-tested patterns to build the next generation of AI-powered applications
