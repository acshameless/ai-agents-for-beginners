# Lista de verificación de calidad

Antes de entregar un cuaderno:

- Ejecutarlo de arriba a abajo al menos una vez (o tanto como permita el entorno).
- Asegurar que las celdas iniciales establezcan todo el estado requerido; evitar el estado oculto de ejecuciones previas.
- Mantener las salidas ordenadas. Evitar salidas enormes cuando un breve resumen sea suficiente.
- Preferir tablas pequeñas, métricas clave o salidas breves.
- Mantener la narración fácil de ojear. Usar encabezados y viñetas cortas, y evitar párrafos largos.
- Dejar TODOs útiles solo cuando sea necesario, y etiquetarlos claramente.
- Si la ejecución no es posible, indicar el riesgo y cómo validar localmente.
