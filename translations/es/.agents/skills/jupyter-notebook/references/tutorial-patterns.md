# Patrones de tutoriales

Utiliza esta estructura para enseñanza y guías paso a paso:

- Audiencia, requisitos previos y objetivos de aprendizaje: indica para quién es, enumera lo que ya deberían saber y expresa qué podrán hacer al final.
- Esquema: proporciona un esquema numerado breve para que los lectores puedan ojearlo.
- Flujo paso a paso: combina una breve explicación en markdown con una pequeña celda de código que se ejecute por sí sola y una breve interpretación del resultado.
- Ejercicios: incluye al menos un ejercicio que refuerce el concepto clave y proporciona una plantilla de respuesta en la siguiente celda.
- Errores comunes y extensiones: señala un error común y cómo solucionarlo, y sugiere una extensión opcional para lectores curiosos.
