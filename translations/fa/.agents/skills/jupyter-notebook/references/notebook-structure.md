# ساختار نوت‌بوک

نوت‌بوک‌های Jupyter اسناد JSON با ساختار سطح بالا زیر هستند:

- `nbformat` و `nbformat_minor`
- `metadata`
- `cells` (یک فهرست از سلول‌های markdown و کد)

هنگام ویرایش فایل‌های `.ipynb` به‌صورت برنامه‌ای:

- مقدار `nbformat` و `nbformat_minor` را از قالب حفظ کنید.
- `cells` را به‌صورت یک فهرست مرتب نگه دارید؛ ترتیب را تغییر ندهید مگر به‌صورت عمدی.
- برای سلول‌های کد، مقدار `execution_count` را وقتی نامعلوم است روی `null` قرار دهید.
- برای سلول‌های کد، `outputs` را هنگام ایجاد ساختار اولیه به یک فهرست خالی تنظیم کنید.
- برای سلول‌های markdown، `cell_type="markdown"` و `metadata={}` را حفظ کنید.

اسکلت‌بندی از قالب‌های بسته‌بندی‌شده یا `new_notebook.py` (برای مثال، `$CODEX_HOME/skills/jupyter-notebook/scripts/new_notebook.py`) را به نوشتن دستی JSON خام نوت‌بوک ترجیح دهید.
