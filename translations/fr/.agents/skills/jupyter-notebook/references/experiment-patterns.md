# Modèles d'expérimentation

Utilisez cette structure pour le travail exploratoire et expérimental:

- Titre et objectif: énoncez la question et les critères de réussite.
- Configuration et reproductibilité: n'importez que ce dont vous avez besoin, définissez une graine tôt, et conservez la configuration dans une seule cellule courte.
- Plan: énumérez les hypothèses, les sweeps et les métriques avant d'exécuter le code.
- Base minimale: commencez par l'exemple exécutable le plus petit et confirmez qu'il s'exécute de bout en bout avant d'ajouter de la complexité.
- Résultats et notes: résumez les conclusions en markdown près du code pertinent et enregistrez les métriques clés dans un petit dictionnaire ou une structure de type tableau.
- Étapes suivantes: décidez de continuer, pivoter ou arrêter, et consignez les idées de suivi sous forme de courtes puces.
