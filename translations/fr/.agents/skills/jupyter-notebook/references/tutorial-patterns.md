# Modèles de tutoriel

Utilisez cette structure pour l'enseignement et les démonstrations :

- Public cible, prérequis et objectifs d'apprentissage : indiquer pour qui c'est, énumérer ce qu'ils doivent déjà connaître et préciser ce qu'ils seront capables de faire à la fin.
- Plan : fournir un bref plan numéroté afin que les lecteurs puissent parcourir rapidement.
- Flux étape par étape : associer une courte explication en markdown avec une petite cellule de code qui s'exécute seule et une brève interprétation du résultat.
- Exercices : inclure au moins un exercice qui renforce le concept clé et fournir un canevas de réponse dans la cellule suivante.
- Pièges et extensions : signaler une erreur courante et comment la corriger, et proposer une extension optionnelle pour les lecteurs curieux.
