שיתוף פעולה בין סוכנים עם LangGraph להסקת מסקנות מובנית

חיפוש וקטורי לשאיבת ידע עם תמיכה מובנית בהטמעות של Cosmos DB
יכולות async של FastAPI לשמירת תגובות מהירות ומדרגיות
בסוף מושב זה יהיה לך מתווה ברור לבניית ופריסת יישומים רב-סוכניים מדרגיים, מבוססי ענן, שמנצלים את כוחו של AI מודרני ותשתיות ענן. בין אם אתה מהנדס AI, אדריכל ענן, או מפתח Python, ההרצאה תספק לך תובנות מעשיות ודפוסים מנוסים לבניית הדור הבא של יישומי AI.
