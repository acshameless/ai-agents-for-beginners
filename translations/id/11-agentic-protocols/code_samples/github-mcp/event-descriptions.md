## Nama Acara: API Multi-Agen dengan LangGraph dan Azure Cosmos DB

## Deskripsi

Meningkatnya aplikasi AI multi-agen mengubah cara kita membangun sistem cerdas - tetapi bagaimana Anda merancangnya untuk skala dan kinerja dunia nyata? Dalam sesi ini, kita akan menyelami aplikasi multi-agen kelas produksi yang dibangun dengan LangGraph untuk orkestrasi agen, FastAPI untuk lapisan API, dan Azure Cosmos DB sebagai tulang punggung untuk manajemen status, penyimpanan vektor, dan data transaksional.

Melalui walkthrough kode yang rinci, Anda akan melihat bagaimana merancang dan mengimplementasikan alur kerja yang digerakkan agen yang mengintegrasikan retrieval-augmented generation (RAG), persistensi memori, dan transisi status dinamis. Kami akan membahas:

Kolaborasi agen dengan LangGraph untuk penalaran terstruktur
Penyimpanan riwayat obrolan real-time menggunakan Azure Cosmos DB - basis data yang sama yang mendukung riwayat obrolan di ChatGPT, aplikasi agen AI dengan pertumbuhan tercepat dalam sejarah
Pencarian vektor untuk pengambilan pengetahuan dengan dukungan embeddings native dari Cosmos DB
Kemampuan async FastAPI untuk menjaga interaksi tetap responsif dan dapat diskalakan
Di akhir sesi ini, Anda akan memiliki cetak biru yang jelas untuk membangun dan menerapkan aplikasi multi-agen cloud-native yang dapat diskalakan yang memanfaatkan kekuatan AI modern dan infrastruktur cloud.
