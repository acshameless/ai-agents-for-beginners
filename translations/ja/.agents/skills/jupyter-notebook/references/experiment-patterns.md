# 実験パターン

探索的および実験的な作業にはこの構成を使用してください：

- タイトルと目的: 質問と成功基準を明確に述べる。
- セットアップと再現性: 必要なものだけをインポートし、早い段階でシードを設定し、設定は短い1つのセルにまとめる。
- 計画: コード実行前に仮説、スイープ、および指標を列挙する。
- 最小ベースライン: 実行可能な最小の例から始め、複雑化する前にエンドツーエンドで動作することを確認する。
- 結果とメモ: 関連するコードの近くにマークダウンで所見を要約し、主要な指標を小さな辞書や表形式の構造に記録する。
- 次のステップ: 継続するか方針を転換するか中止するかを決め、フォローアップのアイデアを短い箇条書きで残す。
