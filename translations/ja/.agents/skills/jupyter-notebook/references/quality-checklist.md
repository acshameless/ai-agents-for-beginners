# 品質チェックリスト

ノートブックを納品する前に:

- 上から下まで少なくとも一度は実行してください（または環境が許す限り実行してください）。
- 先頭のセルですべての必要な状態が設定されていることを確認してください; 以前の実行からの隠れた状態を避けてください。
- 出力は整理しておいてください。短い要約で足りる場合は巨大な出力を避けてください。
- 小さな表、主要な指標、または短い出力を優先してください。
- 説明は読み飛ばしやすくしてください。見出しと短い箇条書きを使い、長い段落は避けてください。
- 必要な場合にのみ有用な TODO を残し、それらに明確なラベルを付けてください。
- 実行が不可能な場合は、そのリスクとローカルでの検証方法を明記してください。
