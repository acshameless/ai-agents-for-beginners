# チュートリアルパターン

この構造を教育やウォークスルーに使用してください:

- 対象読者、前提条件、学習目標: 誰のためかを述べ、彼らがすでに知っているべきことを列挙し、最後に何ができるようになるかを明記する。
- 概要: 読者が目を通しやすいように短い番号付きのアウトラインを提供する。
- ステップバイステップの流れ: 短いマークダウンの説明と、単独で実行できる小さなコードセル、そして結果の簡単な解釈を組み合わせる。
- 演習: 主要な概念を強化する少なくとも1つの演習を含め、次のセルに解答の雛形を提供する。
- 落とし穴と拡張: 一つのよくある間違いとその修正方法を指摘し、好奇心のある読者向けに一つのオプションの拡張案を提案する。
