# Notebook ဖွဲ့စည်းပုံ

Jupyter notebooks သည် ဒီလို အထွေထွေ ပုံစံရှိသော JSON စာရွက်များ ဖြစ်ပါသည်။

- `nbformat` and `nbformat_minor`
- `metadata`
- `cells` (markdown နှင့် code cell များပါရှိသော စာရင်း)

`.ipynb` ဖိုင်များကို ပရိုဂရမ်မှတဆင့် တည်းဖြတ်ရာတွင် -

- Template မှာ ဖြစ်နေသလို `nbformat` နှင့် `nbformat_minor` ကို ထိန်းသိမ်းပါ။
- `cells` ကို အစီအစဉ်ထားသော စာရင်းအဖြစ် ထားပါ; ရည်ရွယ်ချက်ရှိမှသာ အပြန်အလှန် စိစစ် ပြောင်းလဲပါ။
- code cell များအတွက် `execution_count` ကို မသိနိုင်သောအခါ `null` သို့ သတ်မှတ်ပါ။
- code cell များအတွက် scaffolding ပြုလုပ်ရာတွင် `outputs` ကို ဗလာ စာရင်းအဖြစ် သတ်မှတ်ပါ။
- markdown cell များအတွက် `cell_type="markdown"` နှင့် `metadata={}` ကို ထိန်းထားပါ။

raw notebook JSON ကို ကိုယ်တိုင် လက်ဖြင့် ရေးသားရာတွင် အစား bundled templates သို့မဟုတ် `new_notebook.py` (ဥပမာ `$CODEX_HOME/skills/jupyter-notebook/scripts/new_notebook.py`) မှ scaffolding ကို အသုံးပြုရင်း ဦးစားပေးပါ။
