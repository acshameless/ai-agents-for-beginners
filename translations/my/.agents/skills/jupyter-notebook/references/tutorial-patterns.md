# သင်ခန်းစာပုံစံများ

ဤဖွဲ့စည်းမှုကို သင်ကြားရေးနှင့် လမ်းလျှောက်ချက်များအတွက် အသုံးပြုပါ:

- ပရိသတ်၊ လိုအပ်သော မူလအသိပညာများနှင့် သင်ယူရမည့် ရည်မှန်းချက်များ: ဘယ်သူများအတွက်ဖြစ်သည်ကို ပြောပါ၊ သူတို့ အရင်တည်းက သိထားရမည့် အရာများကို စာရင်းပေးပါ၊ အဆုံးတွင် ဘာများကို စွမ်းနိုင်မည်ကို ဖော်ပြပါ။
- အကြမ်းဖျင်းအစီအစဉ်: ဖတ်ရှုသူများ အမြန်ကြည့်နိုင်ရန် နံပါတ်စဉ်တိုတစ်ခု ပေးပါ။
- အဆင့်ချင်းစီးရီး: တိုချုပ်သော Markdown ရှင်းလင်းချက်တစ်ခုကို ကိုယ်ပိုင်အလိုအလျောက် လည်ပတ်နိုင်သည့် ကုဒ်ဆဲလ်သေးတစ်ခုနှင့် ရလဒ်၏ တိုတောင်းသော ရှင်းပြချက်တစ်ခုနှင့် တွဲဖက်ပါ။
- လေ့ကျင့်ခန်းများ: အဓိက အယူအဆကို အတည်ပြုသည့် လေ့ကျင့်ခန်း အနည်းဆုံးတစ်ခု ထည့်ပါ၊ နောက်ဆဲလ်တွင် ဖြေကြားရန် မူကြမ်း (answer scaffold) ပေးပါ။
- အတားအဆီးများနှင့် တိုးချဲ့စရာများ: မကြာခဏဖြစ်တတ်သော အမှားတစ်ခုနှင့် ထိုအမှားကို ပြင်ဆင်နည်းကို ဖော်ပြပါ၊ စိတ်ဝင်စားသော ဖတ်ရှုသူများအတွက် ရွေးချယ်စရာ တစ်ခုတိုးချဲ့အယူအဆကို အကြံပြုပါ။
