## ပွဲအမည်: Irresponsible AI Agents

## ဖော်ပြချက်

AI agent များ၏ စိုးရိမ်ရမှုများကို လေ့လာကာ conversational AI ပလက်ဖောင်းများအတွင်း ယုံကြည်စိတ်ချရသော AI လက်တွေ့ကျကျအား ထည့်သွင်းရသည့် စိန်ခေါ်မှုကို ရင်ဆိုင်ကာ သေချာစွာ ဆွေးနွေးကြမည်။ ဤအစည်းအဝေးတွင် irresponsible AI agent များ၏ ဥပမာများကို နက်ရှိုင်းစွာ ဖော်ပြကာ မော်ဒယ်ပျက်ကွက်မှုများ၊ adversarial jailbreaks နှင့် ယုံကြည်မှုနှင့် ကိုက်ညီမှုကို ဖျက်ဆီးနိုင်သော အခြားအန္တရာယ်များကို ပြသပါမည်။

ကျွန်တော်တို့သည် Microsoft ၏ ယုံကြည်စိတ်ချရသော AI အတွက် cutting-edge ကိရိယာများကိုလည်း စူးစမ်းပါမည်၊ ၎င်းတွင် content filters, red teaming များနှင့် အကဲဖြတ်ခြင်းများ ပါဝင်ပြီး—AI agent များကို တာဝန်ရှိစွာ နှင့် မတာဝန်ရှိစွာ လုပ်ဆောင်နေမှုများကို live demo များနှင့် ပြသပေးမည်။

✅ AI agents များကို ယုံကြည်စိတ်ချရစေရန် လိုအပ်သော နည်းနာများ
