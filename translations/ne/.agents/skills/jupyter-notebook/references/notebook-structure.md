# नोटबुक संरचना

Jupyter नोटबुकहरू यी उच्च-स्तरीय संरचनाका साथ JSON दस्तावेजहरू हुन्:

- `nbformat` र `nbformat_minor`
- `metadata`
- `cells` (मार्कडाउन र कोड सेलहरूको सूची)

जब `.ipynb` फाइलहरू प्रोग्राममार्फत सम्पादन गर्दा:

- टेम्पलेटबाट `nbformat` र `nbformat_minor` कायम राख्नुहोस्.
- `cells` लाई क्रमबद्ध सूचीको रूपमा राख्नुहोस्; जानबुझेर नभएसम्म पुनःक्रम नगर्नुहोस्.
- कोड सेलहरूका लागि, जब थाहा नहुँदा `execution_count` लाई `null` सेट गर्नुहोस्.
- कोड सेलहरूका लागि, खाका तयार गर्दा `outputs` लाई खाली सूचीमा सेट गर्नुहोस्.
- मार्कडाउन सेलहरूका लागि, `cell_type="markdown"` र `metadata={}` राख्नुहोस्.

बन्डल गरिएको टेम्पलेटहरू वा `new_notebook.py` (उदाहरणका लागि, `$CODEX_HOME/skills/jupyter-notebook/scripts/new_notebook.py`) बाट खाका बनाउने कार्य प्राथमिकता दिनुहोस्, काँचो नोटबुक JSON लाई हातले लेख्नुको सट्टा.
