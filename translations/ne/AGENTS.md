# AGENTS.md

## परियोजना अवलोकन

यो रिपोजिटरीमा "शुरुवातकर्ताहरूका लागि AI एजेन्टहरू" छ - AI एजेन्टहरू निर्माण गर्न आवश्यक सबै कुरा सिकाउने व्यापक शैक्षिक कोर्स। यो कोर्समा १५ भन्दा बढी पाठहरू छन् जसले आधारभूतहरू, डिजाइन ढाँचा, फ्रेमवर्कहरू, र AI एजेन्टहरूको उत्पादन तैनाथी समेट्छ।

**प्रमुख प्रविधिहरू:**
- Python 3.12+
- अन्तरक्रियात्मक अध्ययनका लागि Jupyter नोटबुकहरू
- AI फ्रेमवर्कहरू: Semantic Kernel, AutoGen, Microsoft Agent Framework (MAF)
- Azure AI सेवाहरू: Microsoft Foundry, Azure AI Agent Service
- GitHub Models Marketplace (नि:शुल्क स्तर उपलब्ध)

**आर्किटेक्चर:**
- पाठ-आधारित संरचना (००-१५+ डिरेक्टोरीहरू)
- प्रत्येक पाठमा छन्: README कागजात, कोड नमूनाहरू (Jupyter नोटबुकहरू), र छविहरू
- स्वचालित अनुवाद प्रणाली मार्फत बहुभाषी समर्थन
- प्रत्येक पाठमा धेरै फ्रेमवर्क विकल्पहरू (Semantic Kernel, AutoGen, Azure AI Agent Service)

## सेटअप कमाण्डहरू

### पूर्वआवश्यकताहरू
- Python 3.12 वा उच्च संस्करण
- GitHub खाता (GitHub Models का लागि - नि:शुल्क स्तर)
- Azure सदस्यता (वैकल्पिक, Azure AI सेवाहरूका लागि)

### प्रारम्भिक सेटअप

1. **रिपोजिटरी क्लोन वा फोर्क गर्नुहोस्:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # वा
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **Python भर्चुअल वातावरण सिर्जना र सक्रिय गर्नुहोस्:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # विन्डोजमा: venv\Scripts\activate
   ```

3. **निर्भरता स्थापना गर्नुहोस्:**
   ```bash
   pip install -r requirements.txt
   ```

4. **परिवेश भेरिएबलहरू सेटअप गर्नुहोस्:**
   ```bash
   cp .env.example .env
   # आफ्नो API कुञ्जीहरू र अन्त बिन्दुहरू सहित .env सम्पादन गर्नुहोस्
   ```

### आवश्यक वातावरण भेरिएबलहरू

**GitHub Models (नि:शुल्क)** का लागि:
- `GITHUB_TOKEN` - GitHub बाट व्यक्तिगत पहुँच टोकन

**Azure AI सेवाहरू** (वैकल्पिक) का लागि:
- `PROJECT_ENDPOINT` - Microsoft Foundry प्रोजेक्ट अन्त बिन्दु
- `AZURE_OPENAI_API_KEY` - Azure OpenAI API कुञ्जी
- `AZURE_OPENAI_ENDPOINT` - Azure OpenAI अन्त बिन्दु URL
- `AZURE_OPENAI_CHAT_DEPLOYMENT_NAME` - चैट मोडेलको तैनाथी नाम
- `AZURE_OPENAI_EMBEDDING_DEPLOYMENT_NAME` - एम्बेडिङ तैनाथी नाम
- `.env.example` मा देखाइएको थप Azure कन्फिगरेसन

## विकास कार्यप्रवाह

### Jupyter नोटबुकहरू चलाउने

प्रत्येक पाठमा विभिन्न फ्रेमवर्कका लागि धेरै Jupyter नोटबुकहरू छन्:

1. **Jupyter सुरु गर्नुहोस्:**
   ```bash
   jupyter notebook
   ```

2. **पाठ डिरेक्टोरीमा जानुहोस्** (जस्तै `01-intro-to-ai-agents/code_samples/`)

3. **नोटबुकहरू खोल्नुहोस् र चलाउनुहोस्:**
   - `*-semantic-kernel.ipynb` - Semantic Kernel फ्रेमवर्क प्रयोग गर्दै
   - `*-autogen.ipynb` - AutoGen फ्रेमवर्क प्रयोग गर्दै
   - `*-python-agent-framework.ipynb` - Microsoft Agent Framework (Python) प्रयोग गर्दै
   - `*-dotnet-agent-framework.ipynb` - Microsoft Agent Framework (.NET) प्रयोग गर्दै
   - `*-azureaiagent.ipynb` - Azure AI Agent Service प्रयोग गर्दै

### विभिन्न फ्रेमवर्कहरूसँग काम गर्ने

**Semantic Kernel + GitHub Models:**
- GitHub खातासँग नि:शुल्क स्तर उपलब्ध
- सिकाइ र प्रयोगका लागि उपयुक्त
- फाइल नमूना: `*-semantic-kernel*.ipynb`

**AutoGen + GitHub Models:**
- GitHub खातासँग नि:शुल्क स्तर उपलब्ध
- बहु-एजेन्ट समन्वय क्षमता
- फाइल नमूना: `*-autogen.ipynb`

**Microsoft Agent Framework (MAF):**
- Microsoft बाट नवीनतम फ्रेमवर्क
- Python र .NET मा उपलब्ध
- फाइल नमूना: `*-agent-framework.ipynb`

**Azure AI Agent Service:**
- Azure सदस्यता आवश्यक
- उत्पादन-तयार विशेषताहरू
- फाइल नमूना: `*-azureaiagent.ipynb`

### निर्भरता

`requirements.txt` बाट प्रमुख प्याकेजहरू:
- `autogen-agentchat`, `autogen-core`, `autogen-ext` - AutoGen फ्रेमवर्क
- `semantic-kernel` - Semantic Kernel फ्रेमवर्क
- `agent-framework` - Microsoft Agent Framework
- `azure-ai-inference`, `azure-ai-projects` - Azure AI सेवाहरू
- `azure-search-documents` - Azure AI खोज एकीकरण
- `chromadb` - RAG उदाहरणका लागि भेक्टर डेटाबेस
- `chainlit` - च्याट UI फ्रेमवर्क
- `browser_use` - एजेन्टहरूका लागि ब्राउजर स्वचालन
- `mcp[cli]` - मोडेल कन्टेक्स्ट प्रोटोकल समर्थन
- `mem0ai` - एजेन्टहरूको लागि मेमोरी व्यवस्थापन
