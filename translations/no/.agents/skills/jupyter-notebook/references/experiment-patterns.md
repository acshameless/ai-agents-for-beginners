# Eksperimentmønstre

Bruk denne strukturen for utforskende og eksperimentelt arbeid:

- Tittel og mål: angi spørsmålet og suksesskriteriene.
- Oppsett og reproduserbarhet: importer bare det du trenger, sett en seed tidlig, og hold konfigurasjonen i én kort celle.
- Plan: list opp hypoteser, sveip og metrikker før du kjører kode.
- Minimal baseline: start med det minste kjørbare eksempelet og bekreft at det kjører ende-til-ende før du legger til kompleksitet.
- Resultater og notater: oppsummer funn i markdown nær relevant kode og registrer nøkkelmetrikker i en liten ordbok- eller tabell-lignende struktur.
- Neste steg: avgjør om du skal fortsette, endre kurs eller stoppe, og noter oppfølgingsideer som korte punkter.
