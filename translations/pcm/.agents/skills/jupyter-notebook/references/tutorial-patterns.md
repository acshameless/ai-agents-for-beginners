# Tutorial Patterns

Use dis structure for teaching and walkthroughs:

- Audience, prerequisites, and learning goals: talk who e be for, list wetin dem suppose sabi before, and talk wetin dem go fit do by di end.
- Outline: give short numbered outline make readers fit quick see.
- Step-by-step flow: join short markdown explanation with small code cell wey dey run by itself and brief explanation of di result.
- Exercises: put at least one exercise wey go reinforce di main idea and give answer scaffold for di next cell.
- Pitfalls and extensions: talk one common mistake and how to fix am, plus suggest one optional extension for readers wey want learn more.
