# Padrões de Tutorial

Use esta estrutura para ensino e demonstrações guiadas:

- Público-alvo, pré-requisitos e objetivos de aprendizagem: diga para quem é, liste o que já devem saber e indique o que serão capazes de fazer no final.
- Esquema: forneça um breve esquema numerado para que os leitores possam percorrer rapidamente.
- Fluxo passo a passo: associe uma curta explicação em markdown com uma pequena célula de código que corra por si só e uma breve interpretação do resultado.
- Exercícios: inclua pelo menos um exercício que reforce o conceito-chave e forneça uma estrutura de resposta na célula seguinte.
- Erros comuns e extensões: indique um erro comum e como corrigi-lo, e sugira uma extensão opcional para leitores curiosos.
