# Lista de verificare a calității

Before delivering a notebook:

- Execută-l de sus în jos cel puțin o dată (sau atât cât permite mediul).
- Asigură-te că celulele inițiale setează toată starea necesară; evită starea ascunsă din rulări anterioare.
- Păstrează ieșirile ordonate. Evită ieșiri uriașe când un scurt rezumat este suficient.
- Preferă tabele mici, indicatori cheie sau ieșiri scurte.
- Păstrează narațiunea ușor de parcurs. Folosește titluri și liste scurte cu puncte, și evită paragrafele lungi.
- Lasă TODO-uri utile doar când este necesar și etichetează-le clar.
- Dacă execuția nu este posibilă, evidențiază riscul și cum se poate valida local.
