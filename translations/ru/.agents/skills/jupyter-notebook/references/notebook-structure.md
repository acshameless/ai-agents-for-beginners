# Структура блокнота

Jupyter-блокноты — это JSON-документы со следующей общей структурой:

- `nbformat` and `nbformat_minor`
- `metadata`
- `cells` (список markdown- и кодовых ячеек)

При программном редактировании файлов `.ipynb`:

- Сохраняйте `nbformat` и `nbformat_minor` из шаблона.
- Сохраняйте `cells` как упорядоченный список; не меняйте порядок, если это не сделано намеренно.
- Для кодовых ячеек устанавливайте `execution_count` в `null`, когда он неизвестен.
- Для кодовых ячеек устанавливайте `outputs` в пустой список при создании заготовки.
- Для markdown-ячейок сохраняйте `cell_type="markdown"` и `metadata={}`.

Предпочитайте создание заготовок из включённых шаблонов или `new_notebook.py` (например, `$CODEX_HOME/skills/jupyter-notebook/scripts/new_notebook.py`) вместо ручного написания сырого JSON блокнота.
