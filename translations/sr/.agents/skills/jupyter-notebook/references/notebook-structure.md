# Структура Jupyter белешке

Jupyter белешке су JSON документи са овом општом структуром:

- `nbformat` и `nbformat_minor`
- `metadata`
- `cells` (листа Markdown и код ћелија)

Када програмски уређујете `.ipynb` датотеке:

- Задржите `nbformat` и `nbformat_minor` из шаблона.
- Држите `cells` као уређену листу; не мењајте редослед осим ако то није намерно.
- За кодне ћелије, подесите `execution_count` на `null` када је непознато.
- За кодне ћелије, поставите `outputs` на празну листу када креирате скелет.
- За Markdown ћелије, задржите `cell_type="markdown"` и `metadata={}`.

Препоручљиво је користити скелет из приложених шаблона или `new_notebook.py` (на пример, `$CODEX_HOME/skills/jupyter-notebook/scripts/new_notebook.py`) уместо ручног писања сировог notebook JSON-а.
