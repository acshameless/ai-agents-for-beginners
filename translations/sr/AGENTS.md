# AGENTS.md

## Преглед пројекта

Овај репозиторијум садржи "AI Агенти за почетнике" - свеобухватни едукативни курс који учи све што је потребно за израду AI агената. Курс се састоји од више од 15 лекција које покривају основе, дизајн шаблоне, фрејмворке и продукцијско имплементирање AI агената.

**Кључне технологије:**
- Python 3.12+
- Jupyter бележнице за интерактивно учење
- AI фрејмворци: Semantic Kernel, AutoGen, Microsoft Agent Framework (MAF)
- Azure AI сервисе: Microsoft Foundry, Azure AI Agent Service
- GitHub Models Marketplace (доступан бесплатни ниво)

**Архитектура:**
- Структура базирана на лекцијама (фасцикле 00-15+)
- Свакa лекција садржи: README документацију, примерe кода (Jupyter бележнице) и слике
- Подршка за више језика путем аутоматског система превођења
- Више опција фрејмворка по лекцији (Semantic Kernel, AutoGen, Azure AI Agent Service)

## Команде за подешавање

### Претпоставке
- Python 3.12 или новији
- GitHub налог (за GitHub Models - бесплатни ниво)
- Azure претплата (опционо, за Azure AI сервисе)

### Почетно подешавање

1. **Клонирајте или направите форк репозиторијума:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # ИЛИ
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **Креирајте и активирајте Python виртуелно окружење:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # На Windows-у: venv\Scripts\activate
   ```

3. **Инсталирајте зависности:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Подесите променљиве окружења:**
   ```bash
   cp .env.example .env
   # Измените .env са вашим API кључевима и крајним тачкама
   ```

### Потребне променљиве окружења

За **GitHub Models (бесплатно)**:
- `GITHUB_TOKEN` - Лични приступни токен са GitHub-а

За **Azure AI сервисе** (опционо):
- `PROJECT_ENDPOINT` - Microsoft Foundry endpoint пројекта
- `AZURE_OPENAI_API_KEY` - Azure OpenAI API кључ
- `AZURE_OPENAI_ENDPOINT` - Azure OpenAI url endpoint
- `AZURE_OPENAI_CHAT_DEPLOYMENT_NAME` - Име деплојмента за модел ћаскања
- `AZURE_OPENAI_EMBEDDING_DEPLOYMENT_NAME` - Име деплојмента за ембеддинге
- Додатне Azure конфигурације као у `.env.example`

## Радни ток развоја

### Покретање Jupyter бележница

Свакa лекција садржи више Jupyter бележница за различите фрејмворке:

1. **Покрените Jupyter:**
   ```bash
   jupyter notebook
   ```

2. **Идите у фасциклу одређене лекције** (нпр. `01-intro-to-ai-agents/code_samples/`)

3. **Отворите и покрените бележнице:**
   - `*-semantic-kernel.ipynb` - Користећи Semantic Kernel фрејмворк
   - `*-autogen.ipynb` - Користећи AutoGen фрејмворк
   - `*-python-agent-framework.ipynb` - Користећи Microsoft Agent Framework (Python)
   - `*-dotnet-agent-framework.ipynb` - Користећи Microsoft Agent Framework (.NET)
   - `*-azureaiagent.ipynb` - Користећи Azure AI Agent Service

### Рад са различитим фрејмворцима

**Semantic Kernel + GitHub Models:**
- Бесплатни ниво уз GitHub налог
- Добро за учење и експериментисање
- Образац имена фајла: `*-semantic-kernel*.ipynb`

**AutoGen + GitHub Models:**
- Бесплатни ниво уз GitHub налог
- Могућности оркестрације више агената
- Образац имена фајла: `*-autogen.ipynb`

**Microsoft Agent Framework (MAF):**
- Најновији Microsoft фрејмворк
- Доступан у Python и .NET варијанти
- Образац имена фајла: `*-agent-framework.ipynb`

**Azure AI Agent Service:**
- Захтева Azure претплату
- Спреман за продукцијско коришћење
- Образац имена фајла: `*-azureaiagent.ipynb`

## Додатне напомене

### Чести проблеми

1. **Неслагање верзије Python-а:**
   - Проверите да користите Python 3.12+
   - Неки пакети не раде на старијим верзијама
   - Користите `python3 -m venv` за експлицитно постављање верзије

2. **Променљиве окружења:**
   - Увек направите `.env` из `.env.example`
   - Не комитујте `.env` фајл (налази се у `.gitignore`)
   - GitHub токен треба одговарајуће дозволе

3. **Конфликти пакета:**
   - Користите свеж виртуелни енвиронмент
   - Инсталирајте зависности из `requirements.txt`, не појединачно
   - Понеке бележнице могу тражити додатне пакете наведене у markdown ћелијама

4. **Azure сервисе:**
   - Azure AI сервисима је потребна активна претплата
   - Понеке функције су регионално ограничене
   - Ограничења бесплатног нивоа се примењују на GitHub Models

## Контекст специфичан за пројекат

### Подршка за више језика

Овај репозиторијум користи аутоматски систем превођења:
- Подржава преко 50 језика
- Преводи су у директоријумима `/translations/<lang-code>/`
- GitHub Actions workflow води рачуна о ажурирањима превода
- Изворни фајлови су на енглеском у корену репозиторијума

### Структура лекција

Свакa лекција прати доследан образац:
1. Видео сличица са линком
2. Писани садржај лекције (README.md)
3. Примери кода у више фрејмворка
4. Циљеви учења и претходни услови
5. Додатни ресурси за учење повезани

### Именовање примера кода

Формат: `<lesson-number>-<framework-name>.ipynb`
- `04-semantic-kernel.ipynb` - лекција 4, Semantic Kernel
- `07-autogen.ipynb` - лекција 7, AutoGen
- `14-python-agent-framework.ipynb` - лекција 14, MAF Python
- `14-dotnet-agent-framework.ipynb` - лекција 14, MAF .NET

### Специјалне фасцикле

- `translated_images/` - Локализоване слике за преводе
- `images/` - Оригиналне слике за енглески садржај
- `.devcontainer/` - Конфигурација развојног контејнера за VS Code
- `.github/` - GitHub Actions workflow и шаблони

### Зависности

Кључни пакети из `requirements.txt`:
- `autogen-agentchat`, `autogen-core`, `autogen-ext` - AutoGen фрејмворк
- `semantic-kernel` - Semantic Kernel фрејмворк
- `agent-framework` - Microsoft Agent Framework
- `azure-ai-inference`, `azure-ai-projects` - Azure AI сервисе
- `azure-search-documents` - Azure AI претрага интеграција
- `chromadb` - Векторска база података за RAG примере
- `chainlit` - Chat UI фрејмворк
- `browser_use` - Аутоматизација претраживача за агенте
- `mcp[cli]` - Подршка за Model Context Protocol
- `mem0ai` - Управљање меморијом агената
