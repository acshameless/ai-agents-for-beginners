# நோட்புக் அமைப்பு

ஜூபிட்டர் நோட்புக்குகள் இந்த உயர்தர வடிவத்தை கொண்ட JSON ஆவணங்களாகும்:

- `nbformat` மற்றும் `nbformat_minor`
- `metadata`
- `cells` (ஒரு மார்க்டவுன் மற்றும் கோட் செல்கள் பட்டியல்)

`.ipynb` கோப்புகளை செயற்கைக்கோள் மூலம் திருத்தும் போது:

- டெம்ப்ளேட்டிலிருந்து `nbformat` மற்றும் `nbformat_minor` ஐ பாதுகாக்கவும்.
- `cells` ஐ ஒழுங்கமைக்கப்பட்ட பட்டியலாகவைத்திருங்கள்; திட்டமிட்டவிடாமல் மறுஆரம்பம் செய்யாதீர்கள்.
- கோட் செல்களுக்கு, தெரியாத போது `execution_count` ஐ `null` ஆக அமைக்கவும்.
- கோட் செல்களுக்கு, கட்டமைப்பில் `outputs` ஐ வெற்று பட்டியலாக அமைக்கவும்.
- மார்க்டவுன் செல்களுக்கு, `cell_type="markdown"` மற்றும் `metadata={}` ஆகியவற்றை வைக்கவும்.

கையால் எழுதிய மூல நோட்புக் JSON இன் பதிலாக, பைண்ட்லில் உள்ள டெம்ப்ளேட்டுகளிலிருந்து அல்லது `new_notebook.py` (உதாரணமாக, `$CODEX_HOME/skills/jupyter-notebook/scripts/new_notebook.py`) மூலம் கட்டமைப்பதை விரும்புக.
