# பயிற்சி மாதிரிகள்

Use this structure for teaching and walkthroughs:

- இலக்கு வாசகர்கள், முன்னிலைத் தேவைகள் மற்றும் கற்றல்ப் பொருட்கள்: யாருக்காக இது என்பதைக் கூறுங்கள், அவர்கள் ஏற்கனவே என்ன தெரிந்திருக்க வேண்டும் என்பதைக் பட்டியலிட்டுக் கூறுங்கள், மற்றும் முடிவு வரை அவர்கள் என்ன செய்யக்கூடும் என்பதை விவரிக்கவும்.
- வரைமுறை: வாசகர்கள் சுருக்கமாகத் திருத்தி பார்க்கக்கூடியதாக ஒரு சிறிய எண்ணிக்கையிட்ட வரைபட்டியை வழங்கவும்.
- படி படியாக ஓட்டம்: ஒரு குறுகிய markdown விளக்கத்துடன் தனக்கே இயங்கக்கூடிய ஒரு சிறிய குறியீட்டு செல்லை இணைத்து, முடிவின் ஒரு சுருக்கமான விளக்கத்தையும் கொடுத்து ஜோடிக்கவும்.
- பயிற்சிகள்: குறிக்கக் கூட至少 ஒரு பயிற்சியை உள்ளடக்கவும் அது முக்கியக் கருத்தை வலுப்படுத்த வேண்டும் மற்றும் அடுத்த செல்லில் பதிலுக்கான அடிப்படை வடிவத்தை வழங்கவும்.
- சிக்கல்கள் மற்றும் நீடிப்புகள்: ஒரு பொதுவான பிழையை குறிப்பிடவும் அதை எப்படி சரிசெய்வது என்பதை விளக்கவும், ஆர்வமுள்ள வாசகர்களுக்காக ஒரு விருப்பமான விரிவாக்கத்தை பரிந்துரைக்கவும்.
