# AGENTS.md

## Project Overview

இந்த ரெப்பொசிட்டரி "தொடக்கக் கற்றலுக்கான AI முகவர்கள்" என்பவற்றை கொண்டுள்ளது - AI முகவர்களை உருவாக்க தேவையான அனைத்தையும் கற்பிக்கும் விரிவான கல்வி பாடநெறி. இந்த பாடநெறியில் அடிப்படைகள், வடிவமைப்பு மாதிரிகள், கட்டமைப்புகள் மற்றும் AI முகவர்களின் உற்பத்தி நிறுவல் போன்ற 15+ பாடங்கள் உள்ளன.

**முக்கிய தொழில்நுட்பங்கள்:**
- Python 3.12+
- தன்மையாக கற்றலுக்கான Jupyter குறிப்புவழிகள்
- AI கட்டமைப்புகள்: Semantic Kernel, AutoGen, Microsoft Agent Framework (MAF)
- Azure AI சேவைகள்: Microsoft Foundry, Azure AI Agent Service
- GitHub Models சந்தை (இலவச நிலை கிடைக்கும்)

**கட்டமைப்பு:**
- பாடங்கள் அடிப்படையிலான அமைப்பு (00-15+ அடுக்குகள்)
- ஒவ்வொரு பாடத்திலும் உள்ளது: README ஆவணம், குறியீட்டு எடுத்துக்காட்டுகள் (Jupyter குறிப்புவழிகள்), படங்கள்
- தானாக மொழி மாற்றம் செய்யும் பல மொழி ஆதரவு
- ஒவ்வொரு பாடத்திற்கும் பல கட்டமைப்பு விருப்பங்கள் (Semantic Kernel, AutoGen, Azure AI Agent Service)

## Setup Commands

### Prerequisites
- Python 3.12 அல்லது அதற்கு மேல்
- GitHub கணக்கு (GitHub Models - இலவச நிலைக்கு)
- Azure சந்தா (விருப்பம், Azure AI சேவைகளுக்கு)

### Initial Setup

1. **ரெப்பொசிடரி கிளோன் அல்லது ஃபோர்க் செய்யவும்:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # அல்லது
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **Python virtual वातावरण உருவாக்கி செயல்படுத்தவும்:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # விண்டோஸில்: venv\Scripts\activate
   ```

3. **தேவையான (dependencies) கோடுகளை நிறுவவும்:**
   ```bash
   pip install -r requirements.txt
   ```

4. **சூழல் மாறிலிகள் அமைக்கவும்:**
   ```bash
   cp .env.example .env
   # உங்கள் API விசைகள் மற்றும் முடிவெண்களை கொண்டு .env ஐத் தொகுக்கவும்
   ```

### தேவைப்படும் சூழல் மாறிலிகள்

**GitHub Models (இலவசம்)**க்கு:
- `GITHUB_TOKEN` - GitHub-ல் இருந்து தனிப்பட்ட அணுகல் குறியீடு

**Azure AI சேவைகள்** (விருப்பத்திற்காக):
- `PROJECT_ENDPOINT` - Microsoft Foundry திட்ட முகவரி
- `AZURE_OPENAI_API_KEY` - Azure OpenAI API விசை
- `AZURE_OPENAI_ENDPOINT` - Azure OpenAI முகவரி URL
- `AZURE_OPENAI_CHAT_DEPLOYMENT_NAME` - உரையாடல் மாடல் நிறுவல் பெயர்
- `AZURE_OPENAI_EMBEDDING_DEPLOYMENT_NAME` - தொகுத்தல் முதல் நிறுவல் பெயர்
- `.env.example` கோப்பில் காட்டப்பட்ட கூடுதல் Azure அமைப்புகள்

## Development Workflow

### Jupyter குறிப்புவழிகளை இயக்குதல்

ஒவ்வொரு பாடத்திலும் வேறுபட்ட கட்டமைப்புகளுக்கான பல Jupyter குறிப்புவழிகள் உள்ளன:

1. **Jupyter ஐ துவக்கவும்:**
   ```bash
   jupyter notebook
   ```

2. **ஒரு பாடத் தொகுதியை தேர்ந்தெடுக்கவும்** (உதாரணம்: `01-intro-to-ai-agents/code_samples/`)

3. **குறிப்புவழிகளை திறந்து இயக்கவும்:**
   - `*-semantic-kernel.ipynb` - Semantic Kernel கட்டமைப்பு கொண்டு
   - `*-autogen.ipynb` - AutoGen கட்டமைப்பு கொண்டு
   - `*-python-agent-framework.ipynb` - Microsoft Agent Framework (Python)
   - `*-dotnet-agent-framework.ipynb` - Microsoft Agent Framework (.NET)
   - `*-azureaiagent.ipynb` - Azure AI Agent சேவை கொண்டு

### வெவ்வேறு கட்டமைப்புகளுடன் வேலை செய்தல்

**Semantic Kernel + GitHub Models:**
- GitHub கணக்குடன் இலவச நிலை கிடைக்கும்
- கற்றலும் பரிசோதனையும் நல்லதை
- கோப்பு வடிவம்: `*-semantic-kernel*.ipynb`

**AutoGen + GitHub Models:**
- GitHub கணக்குடன் இலவச நிலை கிடைக்கும்
- பல முகவர் ஒருங்கிணைப்புக் திறன்கள்
- கோப்பு வடிவம்: `*-autogen.ipynb`

**Microsoft Agent Framework (MAF):**
- Microsoft இன் புதிய கட்டமைப்பு
- Python மற்றும் .NET ஆக கிடைக்கும்
- கோப்பு வடிவம்: `*-agent-framework.ipynb`

**Azure AI Agent சேவை:**
- Azure சந்தா தேவை
- உற்பத்தி தயாரான அம்சங்கள்
- கோப்பு வடிவம்: `*-azureaiagent.ipynb`

## Pull Request Guidelines

1. **உங்கள் மாற்றங்களை சோதிக்க:**

2. **ஆவண மேம்பாடுகள்:**
   - புதிய கருத்துகளைச் சேர்க்கவிருந்தால் README.md புதுப்பிக்கவும்
