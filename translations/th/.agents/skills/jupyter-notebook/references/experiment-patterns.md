# รูปแบบการทดลอง

ใช้โครงสร้างนี้สำหรับงานสำรวจและงานทดลอง:

- ชื่อเรื่องและวัตถุประสงค์: ระบุคำถามและเกณฑ์ความสำเร็จ.
- การตั้งค่าและการทำซ้ำได้: นำเข้าเฉพาะสิ่งที่คุณต้องการ, กำหนด seed ตั้งแต่ต้น, และเก็บการกำหนดค่าไว้ในเซลล์สั้นๆ เพียงหนึ่งเซลล์.
- แผน: ระบุสมมติฐาน, การทดลองแบบสวีป, และเมตริกก่อนรันโค้ด.
- ฐานขั้นต่ำ: เริ่มจากตัวอย่างที่รันได้เล็กที่สุดและยืนยันว่ามันทำงานครบตั้งแต่ต้นจนจบก่อนเพิ่มความซับซ้อน.
- ผลลัพธ์และบันทึก: สรุปข้อค้นพบเป็น markdown ใกล้โค้ดที่เกี่ยวข้องและบันทึกเมตริกสำคัญในพจนานุกรมขนาดเล็กหรือโครงสร้างคล้ายตาราง.
- ขั้นตอนถัดไป: ตัดสินใจว่าจะดำเนินต่อ, เปลี่ยนทิศทาง, หรือหยุด, และบันทึกแนวคิดติดตามผลเป็นหัวข้อสั้นๆ.
