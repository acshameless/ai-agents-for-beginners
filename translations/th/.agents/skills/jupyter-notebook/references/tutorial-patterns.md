# รูปแบบบทช่วยสอน

ใช้โครงสร้างนี้สำหรับการสอนและการสาธิตทีละขั้น:

- ผู้ชม, ข้อกำหนดเบื้องต้น, และเป้าหมายการเรียนรู้: ระบุว่าเหมาะสำหรับใคร, ระบุสิ่งที่พวกเขาควรจะรู้แล้ว, และระบุสิ่งที่พวกเขาจะสามารถทำได้เมื่อเสร็จสิ้น.
- โครงร่าง: ให้โครงร่างสั้น ๆ แบบมีหมายเลขเพื่อให้ผู้อ่านสามารถสแกนได้.
- ลำดับขั้นตอน: จับคู่คำอธิบายสั้น ๆ ในมาร์กดาวน์กับเซลล์โค้ดขนาดเล็กที่รันได้เอง และการตีความผลลัพธ์สั้น ๆ.
- แบบฝึกหัด: รวมอย่างน้อยหนึ่งแบบฝึกหัดที่เสริมแนวคิดสำคัญ และให้กรอบคำตอบในเซลล์ถัดไป.
- กับดักและการขยาย: ชี้ให้เห็นความผิดพลาดที่พบบ่อยหนึ่งอย่างและวิธีแก้ไข และเสนอส่วนขยายแบบเลือกได้หนึ่งอย่างสำหรับผู้อ่านที่อยากรู้อยากเห็น.
