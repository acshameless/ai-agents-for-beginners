# Mga Pattern ng Eksperimento

Gamitin ang istrukturang ito para sa mapagsiyasat at eksperimental na gawain:

- Pamagat at layunin: ilahad ang tanong at ang mga pamantayan ng tagumpay.
- Pag-setup at kakayahang ulitin: mag-import lamang ng kailangan, mag-set ng seed nang maaga, at ilagay ang konfigurasyon sa isang maikling cell.
- Plano: ilista ang mga hipotesis, mga sweep, at mga metrika bago patakbuhin ang code.
- Pinakamababang baseline: magsimula sa pinakamaliit na pwedeng patakbuhing halimbawa at tiyaking tumatakbo ito mula simula hanggang katapusan bago magdagdag ng komplikasyon.
- Mga resulta at tala: ibuod ang mga natuklasan sa markdown malapit sa kaugnay na code at irekord ang mga pangunahing metrika sa isang maliit na diksyonaryo o istrukturang parang talahanayan.
- Mga susunod na hakbang: magpasya kung magpapatuloy, lilipat ng direksyon, o titigil, at itala ang mga ideyang susundan bilang maiikling puntos.
