# Checklist ng Kalidad

Bago ihatid ang notebook:

- Patakbuhin ito mula itaas hanggang ibaba nang hindi bababa sa isang beses (o hangga't pinapayagan ng kapaligiran).
- Tiyaking itinakda ng mga unang cell ang lahat ng kinakailangang estado; iwasan ang nakatagong estado mula sa mga naunang pagtakbo.
- Panatilihing maayos ang mga output. Iwasan ang napakalaking output kapag sapat ang maikling buod.
- Mas piliin ang maliliit na talahanayan, pangunahing sukatan, o maiikling printout.
- Gawing madaling silipin ang naratibo. Gumamit ng mga heading at maiikling bullet, at iwasan ang mga mahabang talata.
- Mag-iwan lamang ng kapaki-pakinabang na mga TODO kapag kinakailangan, at lagyan ang mga ito ng malinaw na label.
- Kung hindi posible ang pagpapatupad, ilahad ang panganib at kung paano ito mapapatunayan nang lokal.
