# Mga Pattern ng Tutorial

Gamitin ang istrakturang ito para sa pagtuturo at mga walkthrough:

- Mga mambabasa, mga kinakailangan, at mga layunin sa pagkatuto: sabihin kung para kanino ito, ilista ang dapat nang alam nila, at ilahad kung ano ang magagawa nila pagsapit ng katapusan.
- Balangkas: magbigay ng maikling naka-number na balangkas para madaling masilip ng mga mambabasa.
- Daloy na hakbang-hakbang: ipares ang isang maikling paliwanag sa markdown sa isang maliit na code cell na tumatakbo nang mag-isa at isang maikling interpretasyon ng resulta.
- Mga Ehersisyo: isama ang hindi bababa sa isang ehersisyo na nagpapatibay sa pangunahing konsepto at magbigay ng balangkas ng sagot sa susunod na cell.
- Mga karaniwang pagkakamali at mga pagpapalawig: ituro ang isang karaniwang pagkakamali at kung paano ito ayusin, at magmungkahi ng isang opsyonal na pagpapalawig para sa mga mausisang mambabasa.
