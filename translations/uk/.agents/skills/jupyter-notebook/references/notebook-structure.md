# Структура ноутбука

Jupyter notebooks — це JSON-документи з такою загальною структурою:

- `nbformat` та `nbformat_minor`
- `metadata`
- `cells` (список Markdown- та кодових клітинок)

Під час програмного редагування файлів `.ipynb`:

- Зберігайте `nbformat` та `nbformat_minor` із шаблону.
- Зберігайте `cells` як впорядкований список; не змінюйте порядок, якщо це не навмисно.
- Для кодових клітинок встановлюйте `execution_count` у `null`, якщо він невідомий.
- Для кодових клітинок встановлюйте `outputs` як порожній список при створенні заготовки.
- Для Markdown-клітинок зберігайте `cell_type="markdown"` та `metadata={}`.

Надавайте перевагу створенню заготовок із шаблонів, що постачаються в комплекті, або `new_notebook.py` (наприклад, `$CODEX_HOME/skills/jupyter-notebook/scripts/new_notebook.py`) замість ручного створення сирого JSON ноутбука.
