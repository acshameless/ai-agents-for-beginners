# Перелік перевірки якості

Before delivering a notebook:

- Запустіть його з початку до кінця принаймні один раз (або наскільки це дозволяє середовище).
- Переконайтеся, що ранні клітинки встановлюють весь необхідний стан; уникайте прихованого стану від попередніх запусків.
- Тримайте виводи охайними. Уникайте величезних виводів, коли достатньо короткого підсумку.
- Надавайте перевагу малим таблицям, ключовим показникам або коротким виводам.
- Зробіть виклад легким для перегляду. Використовуйте заголовки та короткі марковані пункти, уникайте довгих абзаців.
- Залишайте корисні позначки TODO тільки коли це необхідно і чітко їх маркуйте.
- Якщо виконання неможливе, вкажіть ризики та як перевірити це локально.
