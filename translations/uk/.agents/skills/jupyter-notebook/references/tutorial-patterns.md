# Шаблони підручників

Використовуйте цю структуру для навчання та покрокових демонстрацій:

- Аудиторія, передумови та навчальні цілі: скажіть, для кого це, перелічіть, що вони повинні уже знати, і вкажіть, що вони зможуть робити наприкінці.
- Outline: надайте короткий нумерований план, щоб читачі могли швидко ознайомитися.
- Step-by-step flow: поєднайте коротке пояснення в markdown з невеликою клітинкою з кодом, що виконується самостійно, та короткою інтерпретацією результату.
- Exercises: включіть принаймні одну вправу, яка закріплює ключову концепцію, і надайте каркас відповіді в наступній клітинці.
- Pitfalls and extensions: виокремте одну поширену помилку та як її виправити, і запропонуйте одне необов'язкове розширення для допитливих читачів.
