# تجرباتی نمونے

تجسسی اور تجرباتی کام کے لیے اس ساخت کا استعمال کریں:

- عنوان اور مقصد: سوال اور کامیابی کے معیار بتائیں.
- سیٹ اپ اور قابلِ تکرار: صرف وہی چیزیں درآمد کریں جو آپ کو درکار ہوں، شروع میں سیڈ مقرر کریں، اور کنفیگریشن کو ایک مختصر سیل میں رکھیں.
- منصوبہ: کوڈ چلانے سے پہلے قیاس آرائیاں، سویپس، اور میٹرکس فہرست کریں.
- کم از کم بنیادی مثال: سب سے چھوٹی چلنے والی مثال سے شروع کریں اور پیچیدگی شامل کرنے سے پہلے تصدیق کریں کہ یہ ابتدا سے انتہا تک چلتی ہے.
- نتائج اور نوٹس: متعلقہ کوڈ کے قریب مارک ڈاؤن میں نتائج کا خلاصہ کریں اور اہم میٹرکس کو ایک چھوٹی ڈکشنری یا جدول نما ساخت میں ریکارڈ کریں.
- آئندہ اقدامات: فیصلہ کریں کہ جاری رکھنا ہے، رخ بدلنا ہے، یا روکنا ہے، اور فالو اپ خیالات کو مختصر بلٹس کی صورت میں قلمبند کریں.
