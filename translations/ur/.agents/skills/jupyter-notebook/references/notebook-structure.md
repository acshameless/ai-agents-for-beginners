# نوٹ بک کی ساخت

Jupyter نوٹ بکس JSON دستاویزات ہیں جن کی اعلی سطحی شکل یہ ہے:

- `nbformat` اور `nbformat_minor`
- `metadata`
- `cells` (مارک ڈاؤن اور کوڈ سیلز کی فہرست)

جب پروگرام کے ذریعے `.ipynb` فائلوں میں ترمیم کی جا رہی ہو:

- سانچے سے `nbformat` اور `nbformat_minor` کو برقرار رکھیں.
- `cells` کو ایک ترتیب دی گئی فہرست کے طور پر رکھیں; جب تک ارادہ نہ ہو دوبارہ ترتیب نہ دیں.
- کوڈ سیلز کے لیے، جب نامعلوم ہو تو `execution_count` کو `null` پر سیٹ کریں.
- کوڈ سیلز کے لیے، جب اسکافولڈنگ کی جا رہی ہو تو `outputs` کو خالی فہرست پر سیٹ کریں.
- مارک ڈاؤن سیلز کے لیے، `cell_type="markdown"` اور `metadata={}` کو برقرار رکھیں.

کچے نوٹ بک JSON کو ہاتھ سے تحریر کرنے کے بجائے بنڈل کیے گئے سانچوں یا `new_notebook.py` (مثال کے طور پر، `$CODEX_HOME/skills/jupyter-notebook/scripts/new_notebook.py`) سے اسکافولڈنگ کو ترجیح دیں۔
