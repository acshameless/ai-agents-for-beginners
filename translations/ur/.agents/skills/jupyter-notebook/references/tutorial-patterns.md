# ٹیوٹوریل کے نمونے

تعلیم اور رہنمائی کے لیے اس ڈھانچے کا استعمال کریں:

- سامعین، ضروری شرائط، اور سیکھنے کے مقاصد: بتائیں یہ کس کے لیے ہے، وہ کون سی معلومات پہلے سے جانتے ہونے چاہئیں، اور اختتام تک وہ کیا کرنے کے قابل ہوں گے۔
- خاکہ: ایک مختصر نمبر شدہ خاکہ فراہم کریں تاکہ پڑھنے والے تیزی سے جھانک سکیں۔
- قدم بہ قدم بہاؤ: ایک مختصر مارک ڈاؤن وضاحت کو ایک چھوٹے خود کار چلنے والے کوڈ سیل کے ساتھ جوڑیں اور نتیجے کی مختصر تشریح درج کریں۔
- مشقیں: کم از کم ایک مشق شامل کریں جو کلیدی تصور کو مضبوط کرے اور اگلے سیل میں ایک جواب کے سانچے کی فراہمی کریں۔
- ممکنہ غلطیاں اور توسیعات: ایک عام غلطی کا ذکر کریں اور اسے کیسے درست کریں، اور متجسس قاریوں کے لیے ایک اختیاری توسیع تجویز کریں۔
