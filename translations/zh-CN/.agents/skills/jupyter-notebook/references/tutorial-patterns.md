# 教程模板

在教学和演练中使用此结构：

- 受众、先决条件和学习目标：说明适合谁、列出他们应该已经掌握的知识，并说明在结束时他们将能够做什么。
- 大纲：提供一个简短的编号大纲，便于读者快速浏览。
- 逐步流程：将简短的 Markdown 说明与一个可以独立运行的小代码单元配对，并给出对结果的简要解读。
- 练习：至少包括一个巩固关键概念的练习，并在下一个代码单元中提供答案框架。
- 陷阱与扩展：指出一个常见错误及其修复方法，并为好奇的读者建议一个可选的扩展。
