# 實驗模式

Use this structure for exploratory and experimental work:

- 標題與目標: 陳述問題與成功標準.
- 設定與可重現性: 只匯入所需項目, 盡早設定隨機種子, 並把設定保留在一個簡短的儲存格內.
- 計畫: 在執行程式前列出假設、掃描與指標.
- 最小基線: 從最小可執行範例開始, 並在增加複雜性之前確認其端到端運作正常.
- 結果與註記: 在相關程式碼附近以 markdown 概述發現, 並將關鍵指標記錄在小型字典或類表格結構中.
- 下一步: 決定是繼續、轉向還是停止, 並以簡短的條目記錄後續想法.
