# 教學模式

在教學與示範中使用此結構：

- Audience, prerequisites, and learning goals: 說明適合的對象、列出他們應該已經知道的內容，並說明他們在結束後能夠做到什麼。
- Outline: 提供一個簡短的編號大綱，讓讀者可以快速瀏覽。
- Step-by-step flow: 將簡短的 Markdown 說明與可獨立執行的小程式碼單元配對，並附上一個對結果的簡短解釋。
- Exercises: 包含至少一個強化關鍵概念的練習，並在下一個單元提供一個解答範本。
- Pitfalls and extensions: 指出一項常見錯誤及其修正方法，並為好奇的讀者建議一個可選的延伸練習。
