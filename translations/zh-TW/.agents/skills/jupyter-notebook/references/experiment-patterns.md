# 實驗模式

將此結構用於探索性與實驗性的工作：

- 標題與目標：說明問題與成功標準。
- 設定與可重現性：只匯入必要的套件，儘早設置隨機種子，並將設定保存在一個簡短的 cell（儲存格）中。
- 計畫：在執行程式之前列出假設、參數掃描與度量指標。
- 最小基準：從最小可執行的範例開始，並在增加複雜性前確認其能端到端執行。
- 結果與備註：在相關程式碼附近以 Markdown 彙整發現，並將關鍵指標記錄在小型字典或類表格結構中。
- 下一步：決定是繼續、轉向還是停止，並以簡短項目記下後續想法。
