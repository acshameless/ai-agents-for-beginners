# 教學模式

將此結構用於教學與步驟導覽：

- 受眾、先決條件與學習目標：說明適合對象、列出他們應該已知的事項，並說明課程結束時能夠做什麼。
- 大綱：提供一個簡短的編號大綱，讓讀者可以快速瀏覽。
- 步驟式流程：將簡短的 Markdown 說明與一個可獨立執行的小程式碼儲存格配對，並附上對結果的簡短解釋。
- 練習：至少包含一個強化關鍵概念的練習，並在下一個儲存格提供答案架構。
- 陷阱與延伸：指出一個常見錯誤及其修正方法，並為有興趣的讀者建議一個可選的延伸主題。
